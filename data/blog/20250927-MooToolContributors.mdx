# 今天成为了 MooTool 贡献者

好久不见！已经有一段时间没更新我的博客了！

其实这段时间我的本职工作非常的忙，忙得连写博客都做不了。

直到最近，在GitHub上发现了一个开源项目，这个项目叫MooTool，这是一个基于Java开发的工具

契机是什么呢？契机是我在其他的 Tool 工具中有使用 Text Diff（文本差异比较）的一个功能。

但是这个项目没有，正好我有兴趣开始为开源做贡献，我当然知道这些工具都有现成的。但是自己完成的话也不为是一种满足自己的成就感。

希望这个工具能得越来越多人的喜欢。

<Callout emoji="🎉">
  今天正式成为 <strong>MooTool</strong> 的贡献者！这是一个基于 Java 的开源工具，
  我给它补上了我常用的 <strong>Text Diff（文本差异比较）</strong> 能力。
  希望它能帮助到更多人，也欢迎大家一起来参与贡献。
</Callout>
